# 🎫 高供發票銷售系統 - 使用者端 App

一款 Android 行動應用程式，供使用者瀏覽票券、購買票券、
進行提問與訂單管理，與後端 Web API 串接即時同步資料。

## 🎬 功能展示影片
👉 [https://canva.link/1odpz69pbqo58w6]

## 🛠 使用技術
- 平台：Android（Java）
- 後端串接：ASP.NET Core 8 Web API
- 資料庫：SQL Server / Entity Framework Core
- 驗證機制：QR Code 產生與管理
- 版本控制：Git / GitHub

## ✨ 主要功能
- 使用者註冊／登入
- 票券瀏覽與查詢
- 線上購票與訂單管理
- 提問與客服互動
- QR Code 票券產生（供驗證端掃描）

## 🔗 相關專案
票券驗證端 App：[https://github.com/a47453386/TicketVerifyApp.git]

## 💻 本機執行方式
1. Clone 此專案至 Android Studio
2. 修改 API 連線位址（設定檔中的 Base URL）
3. 確認後端 Web API 已啟動
4. 執行於模擬器或實體裝置
